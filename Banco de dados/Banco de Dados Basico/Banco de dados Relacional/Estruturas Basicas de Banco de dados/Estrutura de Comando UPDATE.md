## **Modificador de Dados**
- O comando UPDATE altera valores existentes em uma tabela:

```
Exemplo: 

UPDATE nome_tabela
SET coluna1 = novo_valro1 , novo_valor2 = novo_valor2
WHERE condicao; 
```
